ReserveIt is a multi-page restaurant reservation web application built with Streamlit. It allows users to browse restaurants, search by preferences, and manage reservations through a simple and interactive interface.

Features:
- Home page for browsing available restaurants
- Search functionality to filter restaurants by criteria such as cuisine, location, and price range
- Reservation history to view and manage past bookings
- Session state management to persist user selections and reservations during interaction
- Custom UI styling with Streamlit to create a cleaner, app-like experience
- Preloaded restaurant dataset with details such as ratings, cuisine types, price ranges, and available time slots

Project Structure:
- app.py (or main file): Initializes the app, session state, and navigation
- pages:
- home.py – Landing page and restaurant browsing
- search.py – Search and filtering interface
- history.py – Reservation history and management
